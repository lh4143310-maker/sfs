# 密码重置页面空白问题 - 快速测试

## 问题
页面打开后完全空白，只显示左下角的二维码。

## 修复内容
修复了 ScrollView 组件的配置问题：
- ✅ 添加 `scrollY` 属性启用垂直滚动
- ✅ 添加 `h-screen` 类设置高度为 100vh
- ✅ 简化外层布局结构
- ✅ 调整 padding 位置

## 技术说明

### 修复前的代码
```tsx
<View className="min-h-screen bg-gradient-subtle flex flex-col items-center p-6">
  <ScrollView className="w-full">
    {/* 内容 */}
  </ScrollView>
</View>
```
**问题**：ScrollView 没有高度，内容无法显示

### 修复后的代码
```tsx
<View className="min-h-screen bg-gradient-subtle">
  <ScrollView scrollY className="h-screen">
    <View className="px-4 py-10">
      {/* 内容 */}
    </View>
  </ScrollView>
</View>
```
**改进**：
- 添加 `scrollY` 启用滚动
- 添加 `h-screen` 设置高度
- 简化布局结构
