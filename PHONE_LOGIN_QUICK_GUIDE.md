# 手机验证码登录 - 快速使用指南

## 功能说明

已成功添加手机验证码登录功能！用户现在可以：
- ✅ 使用手机号+验证码登录
- ✅ 注册时必须绑定手机号
- ✅ 在账号登录和手机登录之间切换

## 快速测试

### 测试手机号登录

1. **打开登录页面**
   ```
   访问：http://localhost:10086/#/pages/login/index
   ```

2. **切换到手机登录**
   - 点击页面上方的"手机登录"选项卡

3. **输入手机号**
   - 输入11位手机号（格式：1[3-9]xxxxxxxxx）
   - 例如：18600737571

4. **获取验证码**
   - 点击"获取验证码"按钮
   - 等待短信验证码（60秒倒计时）

5. **输入验证码**
   - 输入收到的6位验证码

6. **完成登录**
   - 点击"立即登录"按钮
   - 登录成功后自动跳转

### 测试手机号注册

1. **切换到注册模式**
   - 点击"没有账号？立即注册"

2. **选择手机登录**
   - 点击"手机登录"选项卡

3. **填写信息**
   - 手机号：18600737571
   - 用户名：testuser
   - 密码：123456
   - 邮箱：test@example.com（可选）

4. **获取验证码**
   - 点击"获取验证码"
   - 系统会检查手机号是否已注册

5. **输入验证码并注册**
   - 输入验证码
   - 点击"注册账户"
   - 自动登录并跳转

## 功能特点

### 1. 双登录方式
- **账号登录**：用户名 + 密码
- **手机登录**：手机号 + 验证码

### 2. 验证码倒计时
- 点击"获取验证码"后开始60秒倒计时
- 倒计时期间按钮禁用
- 显示剩余秒数

### 3. 智能验证
- 手机号格式验证
- 注册时检查手机号是否已存在
- 验证码正确性验证

### 4. 用户友好
- 清晰的错误提示
- 加载状态显示
- 自动跳转

## 常见问题

### Q1：获取验证码按钮点击无反应
**A**：检查以下几点
- 手机号是否输入正确（11位）
- 是否在倒计时期间
- 网络连接是否正常

### Q2：验证码一直收不到
**A**：可能的原因
- 手机号输入错误
- 短信服务延迟
- 手机拦截了短信

**解决方案**：
- 检查手机号是否正确
- 等待1-2分钟
- 检查垃圾短信箱

### Q3：提示"该手机号已注册"
**A**：说明该手机号已经被使用

**解决方案**：
- 切换到登录模式
- 使用手机号登录
- 或使用其他手机号注册

### Q4：提示"验证码错误"
**A**：验证码输入不正确

**解决方案**：
- 检查验证码是否正确
- 重新获取验证码
- 确保在有效期内使用

## 技术说明

### API 集成
- 使用第三方短信验证码服务
- 通过 Edge Functions 调用 API
- 确保 API 密钥安全

### 数据存储
- 手机号存储在 `user_phones` 表
- 与用户账户关联
- 支持查询和更新

### 安全机制
- 60秒发送间隔限制
- sessionId 机制防止重放攻击
- RLS 策略保护数据

### Edge Functions
1. **send-sms-code**
   - 路径：`supabase/functions/send-sms-code/index.ts`
   - 功能：发送短信验证码

2. **verify-sms-code**
   - 路径：`supabase/functions/verify-sms-code/index.ts`
   - 功能：验证短信验证码

3. **phone-login**
   - 路径：`supabase/functions/phone-login/index.ts`
   - 功能：手机号登录

### 数据库 API
- 路径：`src/db/phone.ts`
- 方法：
  - `sendSmsCode()` - 发送验证码
  - `verifySmsCode()` - 验证验证码
  - `loginWithPhone()` - 手机号登录
  - `bindPhoneToUser()` - 绑定手机号
  - `getUserPhone()` - 获取手机号
  - `checkPhoneExists()` - 检查手机号是否存在

### 前端组件
- 路径：`src/pages/login/index.tsx`
- 状态：
  - `phone` - 手机号
  - `smsCode` - 验证码
  - `sessionId` - 会话ID
  - `loginType` - 登录方式
  - `countdown` - 倒计时
