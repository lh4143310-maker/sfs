## 设计风格统一

### 视觉样式
所有三个页面的预测说明都使用相同的样式：
```tsx
<View className="bg-primary/5 rounded-2xl p-6 border border-primary/20 mb-6">
  <Text className="text-xl text-primary font-bold mb-2 block">预测说明</Text>
  <Text className="text-lg text-muted-foreground leading-relaxed">
    {说明内容}
  </Text>
</View>
```

### 样式特点
- **背景色**：`bg-primary/5` - 主题色5%透明度
- **边框**：`border border-primary/20` - 主题色20%透明度
- **圆角**：`rounded-2xl` - 大圆角
- **内边距**：`p-6` - 充足的内边距
- **标题**：`text-xl text-primary font-bold` - 主题色加粗
- **内容**：`text-lg text-muted-foreground leading-relaxed` - 柔和色调，行距宽松

### 位置统一
所有预测说明都放在：
1. 主要操作按钮（提交/测算按钮）之后
2. 免责声明之前

### 3. 期望管理

- 避免用户产生不切实际的期望
- 提升用户满意度
