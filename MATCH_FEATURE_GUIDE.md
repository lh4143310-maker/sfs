## 使用说明

### 姻缘配对流程
1. 在首页点击"姻缘八字配对"
2. 输入男女双方的姓名、出生日期和时辰
3. 点击"开始合婚测算"
4. 等待 AI 分析（可能需要几秒到几十秒）
5. 查看详细的合婚报告和评分

### 查看历史记录
1. 点击底部导航栏的"记录"
2. 可以看到所有测算记录（包括个人算命和姻缘配对）
3. 点击记录卡片查看详情
4. 点击删除按钮可删除记录

## 注意事项

1. **网络连接**：姻缘配对需要调用 AI 服务，请确保网络连接正常
2. **等待时间**：AI 分析可能需要 10-60 秒，请耐心等待
3. **环境变量**：确保 INTEGRATIONS_API_KEY 已正确配置
4. **错误处理**：如果遇到超时或错误，可以重试

## 技术细节

### Edge Function
- 函数名：`generate_match_report`
- 超时时间：60 秒
- 模型：GPT-4o
- 返回数据：合婚分析文本 + 评分（0-100）

### 数据库
- 表名：`match_records`
- 字段：id, user_id, male_info, female_info, match_result, score, created_at
- 策略：用户只能查看、创建和删除自己的记录

### 前端页面
- 输入页：`/pages/match/index`
- 结果页：`/pages/match-result/index`
- 历史记录：`/pages/history/index`（统一展示）
