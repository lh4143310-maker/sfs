# 算命算法详细说明

## 算法概述

本算命系统基于传统八字命理学，结合《三命通会》《渊海子平》等经典典籍的理论，实现了完全个性化的运势分析。每个用户根据其出生时间、性别、姓名、近期事件等信息，会得到独一无二的算命结果。

## 核心算法流程

### 1. 八字排盘

#### 1.1 四柱计算

**年柱**：根据出生年份计算天干地支
- 以1984年（甲子年）为基准
- 天干：10年一循环（甲乙丙丁戊己庚辛壬癸）
- 地支：12年一循环（子丑寅卯辰巳午未申酉戌亥）

**月柱**：根据出生月份计算
- 地支：正月为寅，依次类推
- 天干：根据年干推算

**日柱**：根据出生日期计算
- 使用儒略日算法（简化版）
- 天干地支各自循环

**时柱**：根据出生时辰计算
- 时辰对应地支：
  - 23-01时：子时
  - 01-03时：丑时
  - 03-05时：寅时
  - 05-07时：卯时
  - 07-09时：辰时
  - 09-11时：巳时
  - 11-13时：午时
  - 13-15时：未时
  - 15-17时：申时
  - 17-19时：酉时
  - 19-21时：戌时
  - 21-23时：亥时
- 时干：根据日干推算

#### 1.2 八字示例

用户输入：
- 出生日期：1990-05-15
- 出生时间：14:30

计算结果：
- 年柱：庚午（1990年）
- 月柱：辛巳（5月）
- 日柱：甲辰（第135天）
- 时柱：辛未（14:30，未时）

完整八字：**庚午 辛巳 甲辰 辛未**

### 2. 五行分析

#### 2.1 天干五行

| 天干 | 五行 |
|------|------|
| 甲、乙 | 木 |
| 丙、丁 | 火 |
| 戊、己 | 土 |
| 庚、辛 | 金 |
| 壬、癸 | 水 |

#### 2.2 地支五行

| 地支 | 五行 |
|------|------|
| 子、亥 | 水 |
| 寅、卯 | 木 |
| 巳、午 | 火 |
| 申、酉 | 金 |
| 辰、戌、丑、未 | 土 |

#### 2.3 五行计分规则

- **天干权重**：15分/个
- **地支权重**：10分/个
- **总分归一化**：转换为百分比（总和100%）

#### 2.4 计算示例

八字：庚午 辛巳 甲辰 辛未

| 柱 | 天干 | 五行 | 分数 | 地支 | 五行 | 分数 |
|----|------|------|------|------|------|------|
| 年 | 庚 | 金 | 15 | 午 | 火 | 10 |
| 月 | 辛 | 金 | 15 | 巳 | 火 | 10 |
| 日 | 甲 | 木 | 15 | 辰 | 土 | 10 |
| 时 | 辛 | 金 | 15 | 未 | 土 | 10 |

**原始分数**：
- 金：15+15+15 = 45
- 木：15
- 水：0
- 火：10+10 = 20
- 土：10+10 = 20

**归一化（总分100）**：
- 金：45%
- 木：15%
- 水：0%
- 火：20%
- 土：20%

### 3. 个性化分析

#### 3.1 事业运势

**分析因素**：
1. **五行属性**：根据最旺的五行推荐适合行业
   - 金旺：金融、科技、机械、法律
   - 木旺：教育、文化、医疗、环保
   - 水旺：贸易、物流、传媒、旅游
   - 火旺：能源、餐饮、娱乐、艺术
   - 土旺：房地产、农业、建筑、咨询

2. **近期事件**：分析用户输入的近期经历
   - 包含"工作"、"事业"、"职场"：强调转折点
   - 包含"困难"、"挑战"、"压力"：强调磨砺
   - 其他：通用鼓励

3. **关注领域**：如果用户选择"事业"，加强相关分析

4. **性别差异**：男命和女命的表述不同

5. **时间预测**：根据当前月份推算吉利月份

**输出示例**：
```
您近期在事业上的变动，正是命中注定的转折点。观您八字【庚午 辛巳 甲辰 辛未】，五行金气较旺，适合从事金融相关行业。您当前正处于事业发展的关键期，建议在7月和10月把握机遇，必有贵人相助。男命以事业为重，宜稳扎稳打，切忌急功近利。三个月内将有重要转机，需提前做好准备。
```

#### 3.2 财运分析

**计算公式**：
```
财运分数 = 金 × 0.4 + 水 × 0.3 + 土 × 0.3
```

**等级划分**：
- 财运分数 > 30：财运亨通
- 财运分数 > 20：财运平稳
- 财运分数 ≤ 20：需谨慎理财

**分析因素**：
1. **五行配置**：金主财，水主流动，土主积累
2. **心情影响**：焦虑→影响判断，兴奋→防止冒进
3. **近期事件**：是否涉及投资、理财、金钱
4. **性别差异**：男命财星、女命财库
5. **投资建议**：根据水的强弱判断偏财运

**输出示例**：
```
您近期在财务方面的关注，说明财运正在转变之中。您当前心态积极，但需防止过于乐观而冒进。综观命局，您当前财运亨通。男命财星得位，可适当进行稳健投资。正财方面较为稳定，偏财运亦有起色，可关注短期投资机会。建议在房产、土地方面多加留意。未来半年内，农历9月为财运高峰期。
```

#### 3.3 婚姻感情

**计算公式**：
```
感情分数 = 火 × 0.35 + 水 × 0.35 + 木 × 0.3
```

**分析因素**：
1. **五行配置**：火主热情，水主情感，木主桃花
2. **性别术语**：乾造/坤造，妻星/夫星
3. **近期事件**：感情、恋爱、婚姻、分手、争吵
4. **心情状态**：孤独、寂寞、幸福、甜蜜
5. **方位推算**：根据五行强弱推算贵人方位
6. **时节推算**：根据五行属性推算有利季节

**输出示例**：
```
您近期在感情方面的经历，正是命中姻缘的显现。您当前心境孤寂，但姻缘自有天定，无需强求。乾造【庚午 辛巳 甲辰 辛未】，妻星明朗，感情运势佳。命带桃花，异性缘佳，男命以财为妻，您的妻星在南方方位或夏季时节较为有利。单身者可在南方方向的场所（如热闹场合）多加留意，热情主动为上策。未来5个月内，感情将有新的进展。
```

#### 3.4 健康养生

**分析逻辑**：
1. **找出最弱五行**：五行失衡易导致健康问题
2. **对应身体部位**：
   - 金：肺、大肠、呼吸系统
   - 木：肝、胆、筋骨
   - 水：肾、膀胱、泌尿系统
   - 火：心、小肠、血液循环
   - 土：脾、胃、消化系统

3. **常见疾病**：根据五行弱项预警
4. **养生建议**：食疗、作息、运动
5. **心情影响**：焦虑伤肝，疲惫伤肾
6. **近期事件**：是否提及健康问题
7. **过旺警示**：五行过旺也需注意

**输出示例**：
```
您近期对健康的关注，说明身体正在发出信号。您近期情绪紧张，易伤肝气，需注意调节心态。观您命局，五行水较弱，需特别注意肾、膀胱、泌尿系统的健康。易出现肾虚、泌尿系统问题等问题。避免熬夜，多饮温水，可食用黑色食物如黑豆、黑芝麻、核桃。同时，五行金过旺，也需注意肺、大肠、呼吸系统的保养，避免呼吸道疾病、皮肤问题。男性在5月至8月期间，尤需注意身体保养。建议每日饮水2000ml以上，配合适度运动，可有效改善体质。
```

#### 3.5 开运指南

**补运原则**：补弱避强
- **补弱**：增强最弱的五行
- **避强**：减少最旺的五行

**开运要素**：
1. **幸运颜色**：根据缺失五行推荐
   - 缺金：白色、金色、银色
   - 缺木：绿色、青色、翠色
   - 缺水：黑色、蓝色、灰色
   - 缺火：红色、紫色、粉色
   - 缺土：黄色、棕色、橙色

2. **幸运数字**：五行对应数字
   - 金：4、9、7
   - 木：3、8、1
   - 水：1、6、4
   - 火：2、7、9
   - 土：5、10、8

3. **幸运方位**：五行对应方向
   - 金：西方
   - 木：东方
   - 水：北方
   - 火：南方
   - 土：中央

4. **开运物品**：
   - 金：金属饰品、铜钱、白水晶
   - 木：木质手串、绿植、绿幽灵
   - 水：黑曜石、水晶球、鱼缸
   - 火：红绳手链、红玛瑙、紫水晶
   - 土：黄水晶、玉石、陶瓷饰品

5. **时间选择**：根据流年推算旺运月份

**输出示例**：
```
张三先生，您的八字【庚午 辛巳 甲辰 辛未】五行缺水，建议通过以下方式开运补运：

【幸运色彩】黑色、蓝色、灰色，可多穿着此类颜色的衣物，或在家居、办公环境中增加相应色调。

【幸运数字】1、6、4，在选择楼层、车牌、手机号等时可优先考虑包含这些数字。

【幸运方位】北方为您的贵人方位，办公桌宜面向此方，有助于提升运势。

【开运物品】建议佩戴黑曜石或水晶球，可放置鱼缸于北方方位。

【时间选择】2026年午年，农历1月、4月、7月、10月为您的旺运月份，重要事宜宜在此期间进行。

【注意事项】避免过多使用白色，少接触金属利器。心诚则灵，顺势而为，自有天佑。
```

## 算法特点

### 1. 完全个性化

每个用户的结果都是独一无二的，基于：
- **出生时间**：决定八字和五行分布
- **性别**：影响称呼和分析角度
- **姓名**：用于个性化称呼
- **近期事件**：影响具体分析内容
- **关注领域**：强化对应维度分析
- **心情状态**：调整建议语气和侧重点

### 2. 逻辑严谨

- 基于传统命理学理论
- 五行生克关系准确
- 天干地支系统完整
- 计算过程可追溯

### 3. 实用性强

- 提供具体的行业建议
- 给出明确的时间节点
- 推荐实际的开运方法
- 关注用户当前状态

### 4. 文化深度

- 使用传统命理术语
- 引用经典典籍理论
- 保持文言文风格
- 体现中华文化底蕴

### 性能优化

- 所有计算在前端完成，无需后端调用
- 算法复杂度：O(1)，计算速度极快
- 结果缓存在数据库，避免重复计算

### 扩展性

- 模块化设计，易于添加新的分析维度
- 可轻松调整权重和算法参数
- 支持添加更多传统命理元素（如大运、流年等）
