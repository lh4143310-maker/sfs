## 任务1：专业罗盘设计改进

### 设计参考
参考了用户提供的传统风水罗盘图片，该罗盘具有以下特点：
- 多层同心圆结构
- 密集的文字信息
- 清晰的分区
- 中心有八卦图案
- 外圈有度数刻度
- 每一圈都有不同的信息（天干地支、二十四山向、纳音等）

### 新罗盘结构（从外到内）

#### 第1层：度数刻度（0-360度）
- **360个刻度线**：每1度一个刻度
- **主刻度**：每10度一个主刻度（更粗更长）
- **度数标签**：每30度显示度数（0°, 30°, 60°, ...）
- **样式**：黑白配色，简洁专业

#### 第2层：二十四山向 + 纳音
- **二十四山向**：从正北子位开始，顺时针排列
  - 四正位（子午卯酉）：红色，字体最大
  - 四隅位（乾坤艮巽）：蓝色，字体中等
  - 其他（天干地支）：黑色，字体较小
- **纳音**：每个山向对应的六十甲子纳音
  - 海中金、炉中火、大林木、路旁土等
  - 显示在山向内侧，字体较小

#### 第3层：分隔圈
- 简单的分隔线，区分不同层次

#### 第4层：十二地支 + 生肖
- **十二地支**：子丑寅卯辰巳午未申酉戌亥
- **圆形标记**：每个地支显示在圆形框内
- **生肖标注**：鼠牛虎兔龙蛇马羊猴鸡狗猪
- **样式**：白底黑字，带边框

#### 第5层：八卦圈
- **八卦符号**：☰☱☲☳☴☵☶☷
- **八卦名称**：乾兑离震巽坎艮坤
- **五行标注**：金木水火土
- **分割线**：八条分割线，形成八个扇形区域
- **样式**：淡色背景，清晰的分区

#### 第6层：中心太极八卦图
- **太极图**：阴阳鱼图案
- **阴中有阳**：黑色区域中的白色圆点
- **阳中有阴**：白色区域中的黑色圆点
- **样式**：经典黑白太极图

#### 罗盘指针
- **红色指针**：指向正北方
- **箭头**：三角形箭头，指示方向
- **样式**：醒目的红色

#### 四个方位标记
- **北（水）**：蓝色圆形标记
- **东（木）**：绿色圆形标记
- **南（火）**：红色圆形标记
- **西（金）**：金色圆形标记

### 技术实现

```typescript
const calculatePosition = (angle: number, distance: number) => {
  const radian = ((angle - 90) * Math.PI) / 180;
  const x = Math.cos(radian) * distance;
  const y = Math.sin(radian) * distance;
  return { x, y };
};
```

## 任务2：多语言支持实现

### 架构设计

#### 1. 语言包结构
```
src/i18n/
├── index.ts          # 语言管理器
├── zh.ts             # 中文语言包
└── en.ts             # 英文语言包
```

#### 2. 语言包内容
**支持的模块**：
- `common` - 通用文本（确定、取消、删除等）
- `tabs` - 底部导航栏
- `home` - 首页
- `input` - 测算输入页
- `result` - 测算结果页
- `consult` - 详细咨询页
- `history` - 历史记录页
- `profile` - 个人中心页
- `wechat` - 微信登录相关

#### 3. 语言管理器功能
```typescript
// 获取当前语言
getCurrentLanguage(): LanguageCode

// 设置语言
setLanguage(lang: LanguageCode): void

// 获取翻译文本
getTranslations(lang?: LanguageCode): Language

// 翻译函数（支持变量替换）
translate(key: string, params?: Record<string, string>, lang?: LanguageCode): string
```

#### 4. React Context
```typescript
interface LanguageContextType {
  language: LanguageCode;
  setLanguage: (lang: LanguageCode) => void;
  t: Language;
}
```

### 使用方法

#### 1. 在组件中使用
```typescript
import { useLanguage } from '@/contexts/LanguageContext';

function MyComponent() {
  const { t, language, setLanguage } = useLanguage();
  
  return (
    <View>
      <Text>{t.home.title}</Text>
      <Text>{t.home.subtitle}</Text>
    </View>
  );
}
```

#### 2. 语言切换
```typescript
// 切换到英文
setLanguage('en');

// 切换到中文
setLanguage('zh');

// 切换语言（中英文互换）
setLanguage(language === 'zh' ? 'en' : 'zh');
```

#### 3. 变量替换
```typescript
// 语言包中定义
deleteMessage: '确定要删除 {name} 的算命记录吗？'

// 使用时
translate('history.deleteMessage', { name: '张三' })
// 输出：确定要删除 张三 的算命记录吗？
```

### 实现的功能

#### 1. 语言切换组件
- **位置**：首页顶部
- **样式**：卡片式按钮，带图标
- **交互**：点击切换语言
- **反馈**：显示当前语言

#### 2. 持久化存储
- 使用 `Taro.setStorageSync` 保存用户选择的语言
- 应用启动时自动加载保存的语言
- 默认语言：中文

#### 3. 全局可用
- 通过 Context 在整个应用中共享
- 在 `app.tsx` 中使用 `LanguageProvider` 包裹
- 所有页面和组件都可以使用 `useLanguage` Hook
