3. **优化的用户流程**：
   - 自动查找已存在用户
   - 新用户自动创建 profile
   - 统一的 session 管理
   - 支持 unionid（跨应用用户识别）

**配置要求**：

需要在配置中心添加以下环境变量：

| 变量名 | 说明 | 获取方式 |
|--------|------|----------|
| `WECHAT_MINIPROGRAM_LOGIN_APP_ID` | 微信小程序 AppID | 微信公众平台 → 开发 → 开发管理 → 开发设置 |
| `WECHAT_MINIPROGRAM_LOGIN_APP_SECRET` | 微信小程序 AppSecret | 微信公众平台 → 开发 → 开发管理 → 开发设置 |

**配置步骤**：
1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入"开发" → "开发管理" → "开发设置"
3. 复制 AppID 和 AppSecret
4. 在本应用的配置中心添加上述两个环境变量
5. 保存配置后，微信登录功能即可使用

**错误提示说明**：
- "微信登录功能未配置"：需要添加环境变量
- "微信登录凭证无效"：code 已过期，请重新登录
- "微信登录频率限制"：调用过于频繁，请稍后再试
- "微信登录凭证已过期"：需要重新获取 code

## 🔧 故障排查

### 姻缘配对失败
1. 检查控制台日志，查看具体错误信息
2. 确认 `INTEGRATIONS_API_KEY` 是否配置
3. 检查网络连接是否正常
4. 尝试重新测算

### 微信登录失败
1. 检查控制台日志，查看环境变量配置状态
2. 确认 AppID 和 AppSecret 是否正确
3. 检查是否在微信小程序环境中
4. 查看微信 API 返回的错误码

### 协议页面无法访问
1. 确认页面路由已在 `app.config.ts` 中注册
2. 检查页面文件是否存在
3. 尝试重新编译项目
