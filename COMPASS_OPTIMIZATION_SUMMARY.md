## 优化内容

### 1. 添加动态旋转效果

**实现方式**：
- 给罗盘主体添加 `animate-rotate-slow` 类
- 使用 CSS animation 实现缓慢旋转
- 旋转速度：60秒一圈（已在 app.scss 中定义）

**效果**：
```tsx
<View className="absolute inset-0 bg-background rounded-full shadow-2xl border-4 border-foreground/20 animate-rotate-slow">
```

**不旋转的元素**：
- 罗盘指针（固定指向北方）
- 四个方位标记（固定位置）
- 添加 `pointer-events-none z-10` 确保在最上层且不影响交互

---

### 2. 优化尺寸控制

**修改前**：
```tsx
<View className="relative w-full max-w-md aspect-square mb-16">
```

**修改后**：
```tsx
<View className="relative w-80 h-80 max-w-[90vw] max-h-[90vw] mb-16">
```

**改进点**：
- 固定尺寸：`w-80 h-80`（320px × 320px）
- 响应式限制：`max-w-[90vw] max-h-[90vw]`
- 确保在小屏幕上不超出屏幕宽度的90%
- 移除 `aspect-square`，使用固定高度

---

### 3. 简化层次结构

**修改前**：6层结构
1. 360个度数刻度 + 度数标签

3. 分隔圈
4. 十二地支 + 生肖
5. 八卦圈 + 五行
6. 中心太极图

**修改后**：5层结构
1. 度数刻度（简化，只保留主刻度）
2. 二十四山向（移除纳音）
3. 十二地支（移除生肖文字）
4. 八卦圈（保留符号和名称）
5. 中心太极图

**移除的元素**：
- ❌ 度数标签（0°, 30°, 60°...）
- ❌ 纳音（海中金、炉中火等）
- ❌ 生肖文字（鼠牛虎兔等）
- ❌ 五行文字（金木水火土）
- ❌ 分隔圈

### 8. 优化边框和透明度

#### 边框宽度
**统一调整**：

- 内层边框：`border-2` → `border` 或 `border-2`

- 边框：`border-foreground/30` → `border-foreground/20`
- 分割线：`bg-foreground/20` → `bg-foreground/15`

**改进效果**：
- 视觉更柔和
- 层次更分明
- 不会过于抢眼
