# 算命大师 - 专业罗盘设计说明

## 罗盘结构（从外到内）

### 第一层：二十四山向（最外圈）
**位置**：距离中心 180px
**内容**：24个方位，每个方位间隔15度
**组成**：
- 八天干：甲、乙、丙、丁、庚、辛、壬、癸（戊己居中不用）
- 十二地支：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥
- 四维卦：乾、坤、艮、巽

**排列顺序（顺时针，从正北子位开始）**：
```
子(0°) → 癸(15°) → 丑(30°) → 艮(45°) → 寅(60°) → 甲(75°) →
卯(90°) → 乙(105°) → 辰(120°) → 巽(135°) → 巳(150°) → 丙(165°) →
午(180°) → 丁(195°) → 未(210°) → 坤(225°) → 申(240°) → 庚(255°) →
酉(270°) → 辛(285°) → 戌(300°) → 乾(315°) → 亥(330°) → 壬(345°)
```

**颜色标注**：
- 四正位（子午卯酉）：红色（text-fire），字号 text-xl
- 四维卦（乾坤艮巽）：主色（text-primary），字号 text-lg
- 其他天干地支：常规色（text-foreground），字号 text-base

### 第二层：度数刻度线
**位置**：距离中心 175px
**内容**：72个刻度线，每5度一个
**样式**：
- 主刻度（每30度）：宽3px，高12px，透明度60%
- 次刻度（每5度）：宽1px，高6px，透明度30%

### 第三层：方位标注环
**位置**：距离中心 145px
**内容**：八个主要方位 + 度数标注

**四正位（红色标注）**：

## 视觉效果

### 动画
1. **八卦盘旋转**：20秒一周，顺时针，无限循环
2. **太极脉动**：2秒周期，透明度 0.7-1.0 变化
3. **平滑过渡**：所有元素支持 transition

### 颜色系统
- **五行色**：
  - 金（Metal）：HSL(45, 100%, 50%) - 黄色
  - 木（Wood）：HSL(120, 60%, 34%) - 绿色
  - 水（Water）：HSL(240, 100%, 10%) - 黑/蓝
  - 火（Fire）：HSL(0, 68%, 42%) - 红色
  - 土（Earth）：HSL(25, 75%, 31%) - 棕色

### 层次感
- 使用多层边框、阴影、透明度营造立体感
- 外圈静止，中圈旋转，形成动静对比
- 渐变背景增强空间感

## 技术实现

### 位置计算
```typescript
const calculatePosition = (angle: number, distance: number) => {
  const radian = ((angle - 90) * Math.PI) / 180;
  const x = Math.cos(radian) * distance;
  const y = Math.sin(radian) * distance;
  return { x, y };
};
```

### 响应式
- 罗盘尺寸：384px × 384px (w-96 h-96)
- 适配移动端屏幕
- 保持圆形比例

文王八卦，用于占卜和风水分析。
