# 微信登录测试清单

## 测试前准备

### 1. 环境变量配置检查

登录 Supabase Dashboard，检查以下环境变量是否已配置：

- [ ] `WECHAT_MINIPROGRAM_LOGIN_APP_ID` 已配置
- [ ] `WECHAT_MINIPROGRAM_LOGIN_APP_SECRET` 已配置
- [ ] 环境变量值正确无误（无多余空格）

**如何检查**：
1. 登录 [Supabase Dashboard](https://supabase.com/dashboard)
2. 选择项目
3. 进入 "Settings" → "Edge Functions" → "Secrets"
4. 确认两个变量都存在

---

### 2. 微信小程序配置检查

登录微信公众平台，确认以下信息：

- [ ] 已获取 AppID（小程序ID）
- [ ] 已获取 AppSecret（小程序密钥）
- [ ] AppID 和 AppSecret 与 Supabase 中配置的一致

**如何检查**：
1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入"开发" → "开发管理" → "开发设置"
3. 查看"开发者ID"部分

---

### 3. Edge Function 部署检查

- [ ] Edge Function `wechat_miniapp_login` 已部署
- [ ] 部署版本是最新的（包含增强的日志）

**如何检查**：
1. 在 Supabase Dashboard 中进入 "Edge Functions"
2. 查看 `wechat_miniapp_login` 的部署状态
3. 确认最后部署时间是最近的

---

## 常见错误排查

### 错误 1：环境变量未配置

**日志特征**：
```
❌ [微信登录] 环境变量未配置
```

**错误信息**：
```
微信配置缺失，请联系管理员配置 WECHAT_MINIPROGRAM_LOGIN_APP_ID 和 WECHAT_MINIPROGRAM_LOGIN_APP_SECRET
```

**解决方案**：
- [ ] 在 Supabase Dashboard 中配置环境变量
- [ ] 等待几秒钟让配置生效
- [ ] 重新测试

---

### 错误 2：微信登录凭证无效

**日志特征**：
```
❌ [微信登录] 微信接口返回错误
❌ [微信登录] 错误码: 40029
```

**错误信息**：
```
微信接口错误 (40029): 微信登录凭证无效，请重试
```

**解决方案**：
- [ ] 检查 AppID 和 AppSecret 是否正确
- [ ] 重新点击登录按钮
- [ ] 确保没有重复调用

---

### 错误 3：Token 验证失败

**日志特征**：
```
✅ [微信登录] 获取到 token: ...
❌ [微信登录] Token 验证错误: ...
```

**错误信息**：
```
登录验证失败: Token has expired or is invalid
```

**解决方案**：
- [ ] 检查 Supabase 认证配置
- [ ] 查看 Edge Function 日志
- [ ] 重新测试

## 快速检查命令

在微信开发者工具的控制台中执行以下命令：

### 检查登录状态
```javascript
Taro.getStorageSync('supabase.auth.token')
```
- 如果返回 token，说明已登录
- 如果返回空，说明未登录

### 清除登录状态
```javascript
Taro.clearStorageSync()
```

### 查看用户信息
```javascript
// 在前端代码中
console.log('当前用户:', user)
console.log('用户资料:', profile)
```
