## 修复详情
1. **加载页面优化**
   - 移除了"AI大师正在为您分析明日运势..."这行文字
   - 保留了简洁的标题"正在预测明日运势"
   - 保留了等待时间提示"预测需要30-60秒，请耐心等待"
   - 视觉更加简洁清爽

2. **大师咨询功能修复**
   - 更新 `fortune_consult` Edge Function 的接口定义
   - 添加 `type` 字段区分不同类型的咨询（fortune、daily_fortune、match、true_love）
   - 为明日运势类型添加专门的字段：`result_text`、`fortune_date`、`input_data`
   - 实现针对明日运势的提示词构建逻辑
   - 明日运势咨询基于运势报告内容，提供个性化建议

3. **按钮逻辑修正**
   - "重新测算"按钮：从 `Taro.reLaunch({ url: '/pages/index/index' })` 改为 `Taro.navigateTo({ url: '/pages/daily-input/index' })`
   - 用户点击后直接返回明日运势输入页面，可以修改信息重新测算
   - 保留了用户之前填写的信息（通过 localStorage 自动回显）
   - "查看历史记录"按钮保持不变，跳转到历史记录页面

## 技术细节
- **Edge Function 接口扩展**: 使用可选字段支持多种咨询类型
- **类型判断**: 通过 `type` 字段区分大盘运势和明日运势
- **提示词优化**: 明日运势咨询更注重基于报告内容的解读和建议
- **导航优化**: 使用 `navigateTo` 而不是 `reLaunch`，保留页面栈，用户可以返回
