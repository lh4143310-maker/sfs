# 算命功能测试示例

## 测试目的

验证算命系统能够根据不同的用户输入生成完全个性化的运势分析结果。

## 测试用例

### 测试用例 1：金命男性，事业焦虑

**输入信息**：
```json
{
  "name": "张三",
  "gender": "男",
  "birthDate": "1990-05-15",
  "birthTime": "14:30",
  "recentEvents": "最近工作压力很大，考虑换工作",
  "focusArea": "事业",
  "mood": "焦虑不安"
}
```

**预期八字**：庚午 辛巳 甲辰 辛未

**预期五行分布**：
- 金：45%（庚、辛、辛）
- 火：20%（午、巳）
- 木：15%（甲）
- 土：20%（辰、未）
- 水：0%

**预期结果特征**：
- ✅ 总结中包含用户姓名"张三先生"
- ✅ 总结中包含完整八字"庚午 辛巳 甲辰 辛未"
- ✅ 总结中提到"五行以金为主（45%）"
- ✅ 总结中提到近期工作压力
- ✅ 事业运势推荐金融、科技等金属性行业
- ✅ 事业运势分析提到"近期在事业上的变动"
- ✅ 财运分析提到"男命财星"
- ✅ 健康建议提到"五行水较弱"，建议补水
- ✅ 开运指南推荐黑色、蓝色（补水）
- ✅ 幸运数字为 1、6、4（水的数字）

## 验证要点

### 1. 个性化验证

**必须不同的内容**：
- [ ] 不同用户的八字完全不同
- [ ] 不同用户的五行分布不同
- [ ] 不同用户的行业推荐不同
- [ ] 不同用户的幸运色、幸运数字不同
- [ ] 不同用户的开运物品不同

**必须相同的结构**：
- [ ] 所有结果都包含完整的八字
- [ ] 所有结果都包含五行分布百分比
- [ ] 所有结果都包含五个维度分析
- [ ] 所有结果都包含幸运色和幸运数字

### 2. 输入影响验证

**姓名和性别**：
- [ ] 男性使用"先生"、"乾造"、"妻星"、"男命"
- [ ] 女性使用"女士"、"坤造"、"夫星"、"女命"
- [ ] 姓名正确出现在总结和开运指南中

**出生时间**：
- [ ] 不同的出生日期产生不同的八字
- [ ] 不同的出生时间产生不同的时柱
- [ ] 八字影响五行分布
- [ ] 五行分布影响所有分析结果

**近期事件**：
- [ ] 包含"工作"、"事业"→事业分析提到转折点
- [ ] 包含"投资"、"理财"→财运分析提到财务关注
- [ ] 包含"感情"、"恋爱"→婚姻分析提到姻缘显现
- [ ] 包含"生病"、"健康"→健康分析提到身体信号

**关注领域**：
- [ ] 选择"事业"→事业分析更详细
- [ ] 选择"财运"→财运分析更详细
- [ ] 选择"感情"→婚姻分析更详细
- [ ] 选择"健康"→健康分析更详细

**心情状态**：
- [ ] "焦虑"、"担心"→提示影响判断
- [ ] "兴奋"、"激动"→提示防止冒进
- [ ] "孤独"、"寂寞"→提示姻缘天定
- [ ] "疲惫"、"劳累"→提示注意休息
- [ ] "平静"、"好"→提示运势平稳

### 3. 逻辑一致性验证

**五行补运**：
- [ ] 幸运色对应最弱五行
- [ ] 幸运数字对应最弱五行
- [ ] 开运物品对应最弱五行
- [ ] 幸运方位对应最弱五行

**行业推荐**：
- [ ] 金旺→金融、科技、机械、法律
- [ ] 木旺→教育、文化、医疗、环保
- [ ] 水旺→贸易、物流、传媒、旅游
- [ ] 火旺→能源、餐饮、娱乐、艺术
- [ ] 土旺→房地产、农业、建筑、咨询

**健康建议**：
- [ ] 金弱→注意肺、呼吸系统
- [ ] 木弱→注意肝、筋骨
- [ ] 水弱→注意肾、泌尿系统
- [ ] 火弱→注意心、血液循环
- [ ] 土弱→注意脾胃、消化系统

## 常见问题

### Q1：为什么两个用户的结果看起来相似？

A：如果两个用户的出生时间非常接近（同年同月同日），他们的八字可能相同或相似，导致五行分布接近。但即使八字相同，由于姓名、性别、近期事件、心情等因素不同，最终的分析内容也会有明显差异。

### Q2：如何验证算法的准确性？

A：可以通过以下方式验证：
1. 使用在线八字排盘工具对比八字是否正确
2. 检查五行分布是否符合天干地支的五行属性
3. 验证幸运色、幸运数字是否对应最弱五行
4. 确认近期事件和心情是否影响了分析内容

### Q3：为什么有些分析内容包含随机元素？

A：部分内容（如吉利月份、遇到良缘的具体场所）使用了一定的随机性，这是为了增加算命的神秘感和趣味性。但核心的八字、五行、行业推荐、健康建议等都是基于严格的逻辑计算的。
