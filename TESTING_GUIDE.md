# 🎯 功能测试指南

## 测试 1：AI 生成运势

### 步骤

1. **打开开发者工具**
   - 按 F12
   - 切换到 Console 标签

2. **进行测算**
   - 进入"开始测算"页面
   - 输入测试数据：
     ```
     姓名：张三
     性别：男
     出生日期：1990-05-15
     出生时间：14:30
     近期事件：工作压力大
     关注领域：事业
     当前心情：焦虑
     ```
   - 点击"开始测算"

3. **查看日志**

**✅ 成功标志**：
```
🔮 开始调用 AI 生成详细运势...
🚀 使用原生 fetch 调用 Edge Function...
✅ Edge Function 请求完成: 200
📦 AI 返回数据: {success: true, ...}
✅ AI 生成运势成功！字数统计: {career: 612, wealth: 423, ...}
```

**❌ 失败标志**：
```
❌ AI 生成运势失败: ...
⚠️ AI 生成失败，使用本地算法
```

4. **验证内容**

在结果页面，查看内容长度：
- **AI 成功**：每个维度 300-600 字，总计 2500+ 字
- **AI 失败**：每个维度 100-200 字，总计 740 字

## 测试 2：详细咨询功能

### 步骤

1. **完成测算**
   - 先完成一次测算（参考测试 1）

2. **进入咨询页面**
   - 在结果页面点击"详细咨询"按钮

3. **输入问题**
   ```
   我最近工作压力很大，不知道是否应该换工作？
   感情方面也遇到了一些问题，不知道如何处理。
   ```

4. **提交并观察**
   - 点击"开始咨询"
   - 观察回答是否逐字显示（流式）
   - 等待完整回答（3-10 秒）

5. **验证回答**

**预期效果**：
- ✅ 回答逐字显示（流式）
- ✅ 内容详实（500-1000 字）
- ✅ 结合八字信息（提到"庚午 辛巳 甲辰 辛未"）
- ✅ 针对问题回答（提到"工作压力"、"换工作"、"感情"）
- ✅ 给出具体建议（时间、方位、方法）

**示例回答**：
```
张三先生，观您八字【庚午 辛巳 甲辰 辛未】，命中金旺（45%），金主决断、果敢、执行力...

您近期提到工作压力很大，正在考虑换工作。从命理角度来看，这正是您事业转折的关键时期。
当前正值2026年2月，春季木旺，而您命中缺水（0%），水生木，木克土，这个时期您可能会感到压力倍增...

建议您在农历三月（阳历4月）或农历八月（阳历9月）做出重要决定...

关于感情方面，您命中火占20%，火主热情、激情...
```

6. **测试停止功能**
   - 再次提交一个问题
   - 在回答过程中点击"停止生成"
   - 验证是否立即停止

## 测试 3：对比测试

### 目的

验证 AI 的随机性和个性化

### 步骤

**测试 A：修改性别**
1. 使用相同的出生信息，只修改性别为"女"
2. 对比结果：
   - 称呼应变为"女士"
   - 婚姻感情分析角度应不同

**测试 B：修改出生时间**
1. 使用相同的其他信息，修改出生时间为"08:00"
2. 对比结果：
   - 八字应不同
   - 所有分析内容应完全不同

**测试 C：修改近期事件**
1. 使用相同的基本信息，修改近期事件为"感情不顺"
2. 对比结果：
   - 婚姻感情部分应特别关注这个问题

**测试 D：重复测算**
1. 使用完全相同的信息，测算两次
2. 对比结果：
   - 八字、五行、幸运色应相同
   - 详细分析内容应有差异（AI 随机性）

## 常见问题

### Q1: AI 生成运势失败

**症状**：
```
❌ AI 生成运势失败
⚠️ AI 生成失败，使用本地算法
```

**排查**：
1. 查看 Edge Function 日志
2. 检查网络连接
3. 确认 API Key 是否配置

### Q2: 咨询功能无响应

**症状**：点击"开始咨询"后没有任何反应

**排查**：
1. 查看控制台日志
2. 确认是否有错误信息
3. 检查网络连接

### Q3: 流式显示不流畅

**症状**：回答不是逐字显示，而是一次性出现

**原因**：可能是网络延迟或 API 响应慢

**解决**：等待网络恢复，重试
