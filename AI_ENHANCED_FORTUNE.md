# AI 增强算命系统说明

## 升级概述

在原有的八字命理算法基础上，接入了 **MiniMax 大语言模型 API**，实现了更加详细、个性化、随机性高的运势分析。

## 技术架构

### 混合算法架构

```
用户输入
    ↓
八字排盘（本地计算）
    ↓
五行分析（本地计算）
    ↓
调用 AI 生成详细运势
    ↓
    ├─ 成功 → 返回 AI 生成的详细内容
    └─ 失败 → 降级到本地算法
```

### 核心优势

1. **详细程度提升**：
   - 旧版：每个维度约 100-150 字
   - 新版：每个维度 300+ 字

2. **个性化程度提升**：
   - 旧版：基于模板 + 变量替换
   - 新版：AI 根据完整信息生成独特内容

3. **随机性提升**：
   - 旧版：固定逻辑，相同输入产生相同输出
   - 新版：temperature=0.9，每次生成都有差异

4. **时效性提升**：
   - 旧版：不考虑当前时间
   - 新版：结合当前年月给出时令建议

## 实现细节

### 1. Edge Function: generate_fortune_detail

**位置**: `supabase/functions/generate_fortune_detail/index.ts`

**功能**: 调用 MiniMax API 生成详细运势

**输入参数**:
```typescript
{
  name: string;           // 姓名
  gender: string;         // 性别
  birthDate: string;      // 出生日期
  birthTime: string;      // 出生时间
  bazi: string;           // 八字（如：庚午 辛巳 甲辰 辛未）
  elements: {             // 五行分布
    metal: number;
    wood: number;
    water: number;
    fire: number;
    earth: number;
  };
  recentEvents: string;   // 近期事件
  focusArea: string;      // 关注领域
  mood: string;           // 心情状态
  personality: string;    // 性格特征
}
```

**输出结果**:
```typescript
{
  success: boolean;
  data: {
    career: string;   // 事业运势（300+字）
    wealth: string;   // 财运分析（300+字）
    love: string;     // 婚姻感情（300+字）
    health: string;   // 健康养生（300+字）
    advice: string;   // 开运指南（300+字）
  };
  usage: {            // API 使用统计
    total_tokens: number;
    prompt_tokens: number;
    completion_tokens: number;
  }
}
```

### 2. Prompt 设计

#### System Prompt（系统提示）

```
你是一位精通《三命通会》《渊海子平》《周易》的资深命理大师，拥有40年的实战经验。

你的分析特点：
1. 每个维度至少300字，内容详实具体
2. 结合传统命理术语和现代生活场景
3. 给出具体的时间节点、方位、颜色、数字等建议
4. 语言专业但通俗易懂，有温度有人情味
5. 每次分析都要有独特见解，避免千篇一律
6. 结合当前时间给出时效性建议
7. 根据用户的近期经历和心情状态，给出针对性的指导
```

#### User Prompt（用户提示）

包含完整的用户信息：
- 基本信息（姓名、性别、出生时间、测算时间）
- 八字命盘（四柱八字）
- 五行分布（金木水火土百分比）
- 性格特征
- 近期状况（近期经历、当前心情、关注领域）

以及详细的分析要求：
- 每个维度的具体分析内容
- 必须结合命理原理
- 必须给出可操作的建议
- 必须体现个性化和针对性

### 3. API 配置

**模型**: MiniMax-M2.5

**参数**:
- `temperature`: 0.9（高随机性）
- `max_completion_tokens`: 4000（允许生成长内容）
- `response_format`: json_object（结构化输出）

### 4. 降级策略

如果 AI 调用失败（网络错误、API 限制、返回格式错误等），系统会自动降级到本地算法，确保用户始终能获得结果。

```typescript
try {
  // 尝试调用 AI
  const aiResult = await callAI();
  return aiResult;
} catch (error) {
  console.error('AI 失败，使用本地算法');
  // 降级到本地算法
  return localAlgorithm();
}
```

- **本地算法**: < 100ms

## 常见问题

### Q1: 为什么有时候结果生成很慢？

A: AI 生成需要调用外部 API，响应时间取决于网络状况和 API 负载。通常需要 3-8 秒。如果超时，系统会自动降级到本地算法。

### Q2: AI 生成的内容准确吗？

A: AI 是基于传统命理学原理和大量训练数据生成的，具有一定的参考价值。但命理学本身具有主观性和解释空间，结果仅供参考。

### Q3: 为什么每次测算同一个人，结果都不一样？

A: 这是 AI 增强的特点之一。通过设置 temperature=0.9，每次生成都会有差异，避免千篇一律。但核心的八字、五行、幸运色等基础数据是固定的。

### Q4: 如果 AI 失败了怎么办？

A: 系统有完善的降级策略。如果 AI 调用失败，会自动使用本地算法，确保用户始终能获得结果。

### Q5: 可以关闭 AI 增强吗？

A: 目前不支持手动关闭。但如果 AI 持续失败，系统会自动使用本地算法。未来可以考虑添加用户设置选项。
